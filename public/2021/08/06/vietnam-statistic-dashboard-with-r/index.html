Vietnam statistic dashboard with R

Posted by Hai Vo on Friday, August 6, 2021

In this project, I built an small Shiny (an R framework to build web app) dashboard to visualize general statistic data of Vietnam. The source of data came from Vietnam general statistics office.

The dashboard now hosted on shinyapps.io but it can be deploy on any others platform. You could browser my dashboard source code at this github repo. The repository also had a Docker file to build the dashboard imagine.
